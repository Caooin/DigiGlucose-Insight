# DigiGlucose-Insight
DigiGlucose Insight（数智糖析）是一款以自然语言对话为核心的AI血糖分析助手，作为用户贴身健康管家，依托多专业Agent协同，提供精准、个性化血糖管理服务。用户可通过日常交谈记录血糖、咨询问题、分享情绪，获得自然响应，打破传统工具操作壁垒。 项目采用对话驱动的多Agent架构：前台对话协调Agent为调度中枢，具备意图识别、上下文管理、任务拆解及个性化回复能力；后台四大专业Agent按需响应，数据记录Agent提取信息并入库，即时分析Agent解读数据，教育科普Agent解答疑问，激励Agent提供情感支持。助手融合专业管理与人性化服务，为用户提供全场景血糖健康支持，助力科学控糖。
